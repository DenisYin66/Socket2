## 简单的JavaWechat演示程序；

- 第一步： 编译Server程序，Run StartServer

- 第二步： 编译Client程序，Run ChatClient ，可运行多个客户端

- 第三步： 演示时，多打开几个客户端，前提时打开了Server端

## 注意
  1.在程序中已经创建了 aa/aa bb/bb cc/cc dd/dd  四个用户
  2.本次编程是基于Socket的，可查阅[网络编程相关资料](https://www.bilibili.com/video/av29769334?from=search&seid=13592130097321159385)
